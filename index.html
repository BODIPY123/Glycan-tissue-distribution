项目说明静态网页 · 本地存储
用途记录糖链 / 蛋白 / 细胞 / 实验条件与器官分布之间的经验关联。
方法使用规则相似度与判别力权重进行分布估算，不是经过验证的机器学习模型。
数据所有数据只保存在当前浏览器 localStorage，不会上传服务器。
建议先导入示例 CSV 熟悉格式；正式使用时请记录时间点、给药方式、模型/样本、检测方式、数据单位和数据来源，避免把不可比较的实验混合计算。

项目说明 / README

录入关联规则（因素组合 → 器官分布）

每条规则记录：糖种类 / 糖数量 / 连接方式 / 蛋白种类 / 细胞种类 / 实验条件，及心肝脾肺肾脑数值。因素可选“任意 / 无特定”作通配。建议 20 组以上，并保持时间点、单位、检测方式可比。

批量导入 / 导出

列顺序：糖种类, 糖数量, 连接方式, 蛋白种类, 细胞种类, 心, 肝, 脾, 肺, 肾, 脑, 时间点, 给药方式, 模型/样本, 检测方式, 数据单位, 数据来源, 备注。旧版 11 列 CSV 仍可导入，新增字段会自动留空。Excel 编辑后请「另存为 CSV UTF-8」再导入。
